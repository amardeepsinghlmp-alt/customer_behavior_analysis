# customer_behavior_analysis
# Data Analytics Project

## Overview

This project demonstrates an end-to-end **data analytics workflow** using Python, SQL, and Power BI. The goal is to load and analyze a dataset, clean and transform the data, explore key insights through exploratory data analysis (EDA), run SQL queries for deeper analysis, and present the results using an interactive Power BI dashboard.

The project also includes a written report and a presentation created with Gamma to communicate insights clearly.

---

## Dataset

The dataset used in this project contains business-related records used for analysis and visualization. It includes multiple fields such as categorical and numerical variables that help analyze patterns, trends, and relationships.

The dataset is loaded and processed using Python before being stored and queried in SQL databases.

The following tools and technologies were used in this project:

* **Python** – Data loading, cleaning, and exploratory data analysis
* **Pandas, NumPy, Matplotlib, Seaborn** – Data analysis and visualization
* **SQL (PostgreSQL / MySQL / SQL Server)** – Querying and data analysis

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python using Pandas.
* Verified the structure of the dataset.
* Checked data types, column names, and missing values.

### 2. Exploratory Data Analysis (EDA)

* Examined summary statistics.
* Identified trends and patterns in the data.
* Created visualizations to understand distributions and relationships.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicates.
* Corrected inconsistent data formats.
* Prepared the dataset for analysis.

### 4. SQL Analysis

The cleaned dataset was loaded into SQL databases to perform structured queries.

Key SQL operations included:

* Data filtering and aggregation
* Grouping and summarizing results
* Joining tables
* Extracting insights using analytical queries

Databases used:

* PostgreSQL
* MySQL
* SQL Server

## How to Run the Project

1. **Clone the repository**

```
git clone https://github.com/your-username/data-analytics-project.git
```

2. **Install required Python libraries**

```
pip install pandas numpy matplotlib seaborn
```

3. **Run the Jupyter Notebook**

```
jupyter notebook
```

4. **Load the dataset and execute the analysis**

5. **Run SQL queries**

* Import the cleaned dataset into PostgreSQL / MySQL / SQL Server
* Execute the provided SQL scripts

6. **Open the Power BI dashboard**

* Open the `.pbix` file in Power BI Desktop to view the dashboard
